# Instagram Metrics

Use this folder to keep clean monthly Instagram performance snapshots that can be reused in strategy, planning, creative direction, and monthly reporting.

## Goal

Do not leave Instagram data trapped in screenshots, app views, or memory.

Convert useful metrics into durable monthly snapshot files so the next planning cycle can use real performance feedback.

## Naming

- `YYYY-MM-instagram-metrics-snapshot.md`

## Workflow

1. When insights, screenshots, or exports are available, create or update the current month snapshot.
2. Keep only the metrics and interpretation that are useful for future decisions.
3. Use the latest snapshot as a reference in:
   - `mitozz-instagram-strategist`
   - `mitozz-content-calendar`
   - `mitozz-creatives-director`
   - Jay monthly reporting

## Priority Signals

- discovery reach, especially non-follower reach
- reach by format
- profile visits from discovery posts
- interactions by format or asset
- top-performing content
- signs of stronger or weaker follow conversion

Do not overread one post. Use the snapshot to spot direction, not to force premature conclusions.
