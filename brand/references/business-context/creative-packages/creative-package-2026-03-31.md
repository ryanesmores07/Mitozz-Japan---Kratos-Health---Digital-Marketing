## 1. Story Trust Proof

### Creative Objective

月末の最後に、「プレミアムとは何か」を見た目の豪華さではなく、基準や姿勢としてやさしく着地させる。ブランドの余韻を静かに残す。

### Target Persona

`Healthy Aging Planner`

### Format Decision

`story sequence`

このテーマはストーリーで短く余韻を作るのが合う。説明しすぎず、価値観として印象に残す。

### Creative Direction

温かい信頼感を主役にした `Set 5 + Set 1` の3フレーム。人物を使う場合は advisor 的で落ち着いた存在感にし、ブランド広告ではなく「選び方の美意識」を感じさせる。

伝えるべき感覚:

- 上質さは見た目だけでは決まらない
- 続けたいと思える安心感もプレミアムの一部
- Mitozzは静かな基準を大切にするブランド

避けるもの:

- ラグジュアリー記号の盛りすぎ
- 強い販売誘導
- 説教っぽい正しさ

### Layout / Structure

3フレーム:

1. Frame 1
   `上質さは`
   `見た目だけではない`

2. Frame 2
   `何を大切にしているかが`
   `静かに伝わること。`

3. Frame 3
   `参考になったら保存`

### Copy Direction

見出し:
`上質さは`
`見た目だけではない`

補足:
`基準や姿勢も、選ぶ理由になる。`

CTA:
`参考になったら保存`

トーン:

- warm
- restrained
- reflective

### Visual Direction

Frame 1 は `Set 5` を主役にした trust portrait か、人物を使わない場合は温かな光と余白で同じ空気を再現する。Frame 2 と 3 は `Set 1` の白カード整理を足し、読みやすく着地させる。

見せたい要素:

- 柔らかな光
- 信頼を感じる視線や姿勢
- 余白
- 温度感のある白とスチールブルー

避けたい要素:

- 化粧品広告っぽい艶感
- 病院パンフレット感
- デコラティブすぎる背景

### Brand / Messaging Check

- premium trust-first positioning: ブランド姿勢をやさしく強める
- calm Japanese-premium execution: 短く、静かで、上質
- general-food-safe compliant messaging: 効果表現ではなく価値観表現に留める
