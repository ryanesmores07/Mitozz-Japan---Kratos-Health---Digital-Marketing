# Creative Package - 2026-03-30

Two assets for March 30:

1. Feed carousel
2. Story reinforcement

## 1. Feed Carousel

### Creative Objective

「なんとなく良さそう」ではなく、信頼できるブランドを選ぶ視点をやさしく整える。月末の締めとして、Mitozzのプレミアムさを見た目ではなく姿勢と判断軸で伝える。

### Target Persona

`Healthy Aging Planner`

時間をかけてでも納得して選びたい人。派手な効能訴求より、信頼、姿勢、継続しやすさを見て判断する層。

### Format Decision

`carousel`

このテーマは一枚の印象だけで終えるより、信頼の判断軸を順番に整理した方が説得力が出る。3枚構成で短く、上質に、保存しやすい形にする。

### Creative Direction

「信頼できるサプリ選び」を不安訴求ではなく、静かな判断基準の話として見せる。トーンは月末の安心感。権威的すぎず、やわらかいが甘くない。

伝えるべき感覚:

- プレミアムとは価格や見た目だけではなく、選ぶ理由が整っていること
- 成分やブランドの姿勢を落ち着いて見ることが大切
- Mitozzは急かすブランドではなく、納得して選ぶ人に寄り添うブランド

避けるもの:

- 不安を煽る比較
- 他ブランドを下げる表現
- 強い販売色
- 医療監修風の硬すぎる見せ方

### Layout / Structure

3枚構成:

1. Cover
   `なぜ今`
   `信頼できるサプリ選びが大切なのか`

2. Trust criteria
   `見た目の印象だけでなく`
   `成分 伝え方 ブランド姿勢を見る`

3. End frame
   `納得して選ぶことが`
   `毎日の安心につながる`

### Copy Direction

カバー見出し:
`なぜ今`
`信頼できるサプリ選びが大切なのか`

本文方向:

1.
`見た目の上質さだけでなく、何を大切にしているブランドかを見ることが大切です。`

2.
`成分、伝え方、姿勢まで含めて納得できるか。`

3.
`毎日続けるものだからこそ、信頼できる選び方を。`

CTA:
`ブランドの考え方を読む`

トーン:

- calm
- quietly premium
- reassuring
- mobile-readable

### Visual Direction

主軸は `Set 5 + Set 1`。1枚目は trust portrait の空気感を強め、2枚目と3枚目で white-card 的な整理を足す。人物を使う場合も広告的な笑顔ではなく、落ち着いて判断する知性を感じる表情にする。

見せたい要素:

- 温度の低すぎないスチールライト
- 余白の多い信頼訴求
- portrait と text-card の接続
- 読ませすぎない短い判断軸

避けたい要素:

- corporate medical stock look
- 金ピカの高級感
- 文字量の多い比較表
- サプリEC広告のような値引き感

### Brand / Messaging Check

- premium trust-first positioning: 選ぶ基準と姿勢の話に重心を置いている
- mitochondria-first education: 直接成分説明ではないが、ブランド理解の土台を作る
- calm Japanese-premium execution: 余白と静けさを保ち、月末の安心感に寄せている
- general-food-safe compliant messaging: 効果断定や治療的表現は使わない

## 2. Story Reinforcement

### Creative Objective

フィードの信頼テーマを、ストーリーでより軽く自分ごと化してもらう。ブランドを選ぶ時に何を見るかを考えてもらう入口にする。

### Target Persona

`Healthy Aging Planner`

### Format Decision

`story sequence`

信頼テーマはストーリーで問いにすると反応が取りやすい。重い説明は避け、シンプルな判断軸にする。

### Creative Direction

フィードの内容を短くして、「あなたは何を見る？」という視点に寄せる。静かな参加導線として使う。

### Layout / Structure

3フレーム:

1. `ブランドを信頼する時`
   `何を見ますか？`

2. `見た目だけでなく`
   `成分や姿勢も判断軸に。`

3. `あなたが最初に見る点は？`

### Copy Direction

CTA:
`最初に見る点は？`

トーン:

- light
- premium
- conversational

### Visual Direction

`Set 1` を簡略化して使う。白場多め、短い質問、静かな背景。必要なら frame 1 のみごく薄く `Set 5` の温度感を借りる。

### Brand / Messaging Check

- premium trust-first positioning: 質問ベースで信頼形成につなげる
- calm Japanese-premium execution: フィードと連続性がある
- compliant: 価値判断の話に留め、機能断定をしない

### Nano Banana / Sora Handoff

- `asset_archetype`: `story-reinforcement`
- `visual_intent`:
  - `minimal trust-led story cards`
  - `quiet white-card structure`
  - `simple question-led pacing`
- `selected_image_references`:
  - `brand/references/business-context/visual/reference-pack/working-examples/feed-education-white-card-01.png`
  - `brand/references/business-context/visual/reference-pack/style-anchors/anchor-portrait-trust-01.png`
  - `output/instagram/templates/2026-03-21-approved-pass/set-5-trust-portrait-v01.png`
