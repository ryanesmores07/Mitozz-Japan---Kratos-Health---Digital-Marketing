# Mitozz Carousel Blueprint System

## Purpose

This file translates the Mitozz design system into repeatable slide maps for common carousel lengths.

Use it when building:

- educational carousels
- ingredient explainers
- myth clarification posts
- routine and lifestyle carousels
- authority-led carousel posts

## How To Use

For each post:

1. assign the post type
2. choose the slide count
3. assign a slide role to each frame
4. match each role to the recommended template set
5. vary crop and focal treatment without breaking the system

## Template Set Key

- `Set 1`: Education White Card
- `Set 2`: Editorial Science Layer
- `Set 3`: Luminous Product Hero
- `Set 4`: Real-Life Ritual Lifestyle
- `Set 5`: Warm Trust Portrait

## 3-Slide Blueprint

Best for:

- simple education
- myth clarification
- short ingredient introduction
- story-like micro carousel

Slide map:

1. `Hook`
   - template: `Set 1`
   - goal: stop the scroll with one clear save-worthy idea

2. `Clarification`
   - template: `Set 2`
   - goal: explain the core point simply and cleanly

3. `Soft CTA`
   - template: `Set 3` or simplified `Set 1`
   - goal: end with product relevance or a low-pressure save prompt

## 5-Slide Blueprint

Best for:

- standard education posts
- mitochondria explainers
- ingredient meaning posts
- routine frameworks

Slide map:

1. `Hook`
   - template: `Set 1`
   - goal: strong problem, payoff, or framing statement

2. `Definition`
   - template: `Set 2`
   - goal: explain the concept in the simplest possible terms

3. `Clarification`
   - template: `Set 2`
   - goal: add nuance without raising text density too much

4. `Daily-life translation`
   - template: `Set 4` or light `Set 1`
   - goal: connect the idea to real life, routine, or felt experience

5. `Soft CTA`
   - template: `Set 3`
   - goal: close with product relevance, trust, or save/share behavior

## 7-Slide Blueprint

Best for:

- deeper educational posts
- structured explainers
- topic breakdowns that need a little more pacing

Slide map:

1. `Hook`
   - template: `Set 1`
   - goal: establish the main promise or reframe

2. `Definition`
   - template: `Set 2`
   - goal: explain the base concept

3. `Clarification`
   - template: `Set 2`
   - goal: add the next most useful layer

4. `Daily-life translation`
   - template: `Set 4`
   - goal: reduce abstraction and make the post feel human

5. `Clarification`
   - template: `Set 2`
   - goal: return to the knowledge layer with one more practical point

6. `Proof or trust`
   - template: `Set 5` or `Set 3`
   - goal: add credibility, brand meaning, or premium reassurance

7. `Soft CTA`
   - template: `Set 3`
   - goal: give the post a clean final hold

## 10-Slide Blueprint

Best for:

- pillar content
- major explainer posts
- comparison frameworks
- anchor educational assets worth saving

Rules:

- no slide should carry more than one main idea
- alternate heavier information slides with lighter breathing slides
- use one realistic or human slide before fatigue sets in

Slide map:

1. `Hook`
   - template: `Set 1`
   - goal: strong opening claim, question, or myth break

2. `Definition`
   - template: `Set 2`
   - goal: establish the foundation

3. `Clarification`
   - template: `Set 2`
   - goal: explain the first important nuance

4. `Daily-life translation`
   - template: `Set 4`
   - goal: connect the topic to routine, environment, or daily condition

5. `Clarification`
   - template: `Set 2`
   - goal: continue the logic in a clean way

6. `Clarification`
   - template: `Set 2`
   - goal: complete the deeper educational sequence

7. `Proof or trust`
   - template: `Set 5`
   - goal: slow the pace and restore emotional trust

8. `Product relevance`
   - template: `Set 3`
   - goal: introduce product context without turning into an ad

9. `Recap`
   - template: simplified `Set 1`
   - goal: compress the takeaway into a memorable frame

10. `Soft CTA`
    - template: `Set 3`
    - goal: finish with a premium hold for saving, sharing, or reading more

## Post-Type Adjustments

Educational explainer:

- lean on `Set 1` and `Set 2`
- use `Set 4` only once for relief
- use `Set 3` only at the end

Ingredient explainer:

- use `Set 1` for the cover
- increase `Set 2` usage in the middle
- end with `Set 3`

Lifestyle or routine post:

- lead with `Set 4`
- support with `Set 1` and `Set 2`
- close with `Set 3`

Authority or expert post:

- lead or interrupt with `Set 5`
- support explanation with `Set 1`
- use `Set 3` only if product context is needed

## Design Fatigue Rules

To keep long carousels readable:

- avoid using the same exact card layout twice in a row
- change alignment every 1 to 2 slides
- rotate between close, medium, and airy framing
- use at least one lighter frame by slide 4 in longer posts
- keep the CTA slide visually simpler than the body slides

## Final Check

Before production, confirm:

- the slide map matches the post objective
- each frame has one job only
- the template-set mix feels intentional
- the carousel stays recognizably Mitozz from slide 1 to the end
