# Working Examples

This folder holds explicitly approved generated outputs that are safe to use as working references for future generations.

Approved working examples should:

- reinforce the Mitozz system without freezing it into one rigid template
- act as secondary references after `style-anchors`, not replacements for them
- be used to preserve premium consistency across the grid while still allowing controlled variation

Rules:

- Do not use exploratory or unapproved outputs as guidance.
- Only move an asset here after explicit human approval.
- Use working examples to reinforce brand uniformity, not to clone an existing post.
- Keep variation in crop, focal treatment, text rhythm, and glow/background behavior.
- If this folder is empty, generate from `style-anchors/` plus the visual direction docs only.
