# Source Intake

This folder stores extra inspiration images that are not part of the active style-anchor set.

Use it as a holding area when:

- an image influenced the direction but is too repetitive to become an anchor
- an image is useful for future review but should not guide current generation
- you want to keep the active `style-anchors/` set lean

Do not reference files in this folder in prompt `image_references` unless they are later promoted into `style-anchors/`.
