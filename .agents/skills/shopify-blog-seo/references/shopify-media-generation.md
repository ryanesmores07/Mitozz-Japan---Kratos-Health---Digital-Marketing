# Shopify Media Generation Reference

Use this reference when writing the hero-image prompt for Shopify Magic.

Source: Shopify Help Center page "Media generation in the file editor":
https://help.shopify.com/en/manual/shopify-admin/productivity-tools/shopify-magic/media-generation

## Prompting Rules To Follow

- Use simple natural language.
- Prefer short keywords or short phrases separated by commas.
- Keep prompts at least a few words long.
- Do not write the prompt like a conversation with a chatbot.
- Do not pile on long lists of instructions.
- Describe the scene, setting, style, or lighting directly.
- Add size or scale details when object proportion matters.
- Ground the subject in the scene when relevant so it does not appear to float.

## Shopify-Friendly Prompt Ingredients

When useful, combine a few of these elements:

- location or setting
- lighting
- color or palette
- shot type or camera angle
- style or finish
- subject grounding

Examples of ingredient types from Shopify's guidance:

- location: `indoors`, `outdoors`, `in the city`
- lighting: `warm even light`, `ambient`, `overcast`, `studio lights`
- color: `vibrant`, `muted`, `pastel`
- framing: `front shot`, `closeup`, `from above`

## Good Shape For This Skill

For blog hero images, prefer prompts shaped like:

`subject, setting, lighting, framing, style`

Example pattern:

`open notebook beside herbal tea on wooden desk, soft morning light, front three-quarter view, clean editorial photography`

Avoid:

- long storytelling instructions
- multiple conflicting styles
- requests for rendered text inside the image
- prompt language that sounds like a full paragraph of directions
